# Wiki Schema

## Domain
AI/ML 工具与智能体生态

## Conventions
- File names: lowercase, hyphens, no spaces
- Every wiki page starts with YAML frontmatter
- Use [[wikilinks]] to link between pages (minimum 2 outbound links per page)
- When updating a page, always bump the `updated` date
- Every new page must be added to `index.md` under the correct section
- Every action must be appended to `log.md`
- Language: 中文撰写，技术术语保留英文

## Tag Taxonomy
- agent-framework, mcp, tool-use, multi-agent
- knowledge-base, note-taking, retrieval
- model, fine-tuning, inference, benchmark
- dev-tool, cli, api
- person, company, open-source
- comparison, workflow, tutorial, controversy
